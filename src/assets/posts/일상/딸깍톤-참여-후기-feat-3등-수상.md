## 딸깍톤에 참가하다!
올해 만우절을 기념하여 열린 `바이브코딩 딸깍톤`에 참여하게 되었습니다!
부스트캠프 그룹 스프린트 과정을 함께하며 동고동락했던 든든한 팀원들(`대니`, `제리`, `조이`)과 오랜만에 다시 함께 하게 되었어요. 이번 대회의 핵심은 **'직접 코드를 작성하지 않는 것'**과 **‘재미있고 엉뚱한 아이디어를 구현하는 것‘**이었습니다. 평소처럼 복잡한 로직을 고민하며 코드를 타이핑하는 대신 기획과 AI 프롬프팅(<span class="text-red">**딸깍!**</span>)만으로 짧은 시간 안에 재미있는 아이디어를 구현해 보는 경험을 해보기로 했습니다.
## 아이디어 회의: 진정성이 느껴지는 온라인 반성문
<img src="https://www.notion.so/image/https%3A%2F%2Fprod-files-secure.s3.us-west-2.amazonaws.com%2Fce63075d-cee3-44a7-8d31-efcf5784b0e7%2Fd3142138-c84d-4df8-a23c-22fc0ca85b02%2Fimage.png?table=block&id=367d67b9-9e80-80cc-a765-c775ce237604&cache=v2" alt="image" width="1394" height="1068" loading="lazy" />

이번 해커톤의 메인 키워드는 `만우절`이었고 대회 전 팀별로 `드레스코드 맞춰 입고 오기`라는 특별한 내용이 공지되었어요.
저희 팀은 드레스코드도 정하고 서비스 아이디어도 구체화하기 위해 줌과 피그잼으로 온라인 회의를 진행했어요. 정말 재미있는 다양한 아이디어들이 쏟아졌는데 최종적으로 <span class="text-blue">**진정성 있는 반성을 강제하는(쓰는 사람은 열받는) 온라인 반성문 서비스**</span>를 만들기로 결정했어요.
반성문에 진정성을 담기 위해 저희가 기획한 '킹받는' 기능들중 주요 기능들은 다음과 같아요.
- **키보드 압수:** 오직 마우스 클릭으로만 글자 입력 가능
- **실수란 없다:** 단 한 번의 실수로 한 글자라도 지우면 처음부터 다시 작성
- **랜덤 키보드:** 숫자나 특수문자 자리가 쉴 새 없이 랜덤으로 변경
사용자가 최대한 어렵고 고통스럽게 반성문을 쓰도록 유도하는 것이 목표였으며 여기에 작성한 글자 수, 수정한 횟수 등을 종합해 점수를 측정해 주고 작성을 완료하면 저희가 직접 만든 '참 잘했어요' 도장을 찍어 PDF 파일로 저장할 수 있는 디테일도 추가하기로 했어요.
:::callout 💡 **⚾️ 우리 팀의 드레스코드는?**
여러 아이디어가 오가던 중 마침 해커톤 당일이 '야구 개막 시즌'이라는 점에 착안해 **야구 유니폼**을 입고 가기로 했어요!

아이디어를 실제 서비스로 구현하기 위해 프로그래밍 언어 대신 **자연어**로 AI에게 저희가 원하는 기능을 상세하게 설명했습니다. 결과를 실시간으로 확인하면서 "이 부분을 조금 더 열받게 바꿔줘!", "이건 이렇게 동작하게 해 줘!"라고 요청하는 과정을 팀원들과 함께하니까 직접 코드를 짜는 것과는 또 다른 재미가 있었고 팀원들과 어떻게 하면 사용자를 더 괴롭게(?) 만들지 고민하며 재미있게 진행했던 것 같아요.
어느새 제출 시간이 훌쩍 다가왔고 무사히 최종 결과물을 제출한 뒤 대니가 발표까지 아주 잘해주었어요!!
## 서비스 발표와 결과..
일단 결과부터 말하자면 무료 3등!!! 🥉

사실 발표 시간에 다른 팀들의 결과물을 보면서 '와, 저걸 코딩 없이 기획과 프롬프트만으로 만들었다고?' 싶을 정도로 완성도 높고 기발한 아이디어들이 많아 감탄하고 있었는데요. 그래서 수상은 전혀 예상하지 못했는데 저희 팀 이름이 호명되어 정말 깜짝 놀랐고 기분이 좋았어요.
아마 저희가 발표할 때 반성문 작성자를 괴롭히기 위한 기능들이 소개될 때마다 다른 참가자분들이 공감과 분노(?)를 해주신 덕분이 아닐까 싶네요 ㅎㅎㅎ 저희 서비스의 킹받는 기획 의도를 유쾌하게 봐주시고 소중한 한 표를 던져주신 모든 분들께 다시 한번 감사합니다! ㅎㅎ
